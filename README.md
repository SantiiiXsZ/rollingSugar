# 🍬 Rolling Sugar

Sitio web de recetas de repostería desarrollado de forma colaborativa por el equipo de Rolling Code.

## 🎂 Demo del proyecto

Mira la demo [aquí](https://rollingsugarweb25.netlify.app/)

## 🛠️ Tecnologías y herramientas usadas

- HTML5
- CSS3
- Bootstrap v5.3

## ➡️ Pasos para clonar el repo

En una terminal ejecuta el siguiente comando:

```bash
git clone https://github.com/SantiiiXsZ/rollingSugar.git
```
